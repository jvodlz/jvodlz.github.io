Discord Bot

A simple discord bot
Offering Motivating and Inspirational quotes.

Description

Several APIs were used to create the bot and its functions including Discord, zenquotes.io and stoic-api.herokuapp.com.

The bot detects specific keywords in the chat which in turn determines what quotes would be generated. Both API used for the quotes are in JSON.
